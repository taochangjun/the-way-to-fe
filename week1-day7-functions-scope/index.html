Day7：函数与作用域

这页用几个小实验帮你建立直觉：函数拆分、作用域链、闭包。

建议：打开浏览器 DevTools → Console，一边点按钮一边看输出。

实验 1：闭包（计数器工厂）

关键点：A 和 B 是用同一个函数“工厂”创建的，但它们各自拥有自己的私有 count。

实验 2：作用域链（从内到外找变量）

点击“运行示例”开始。

关键点：变量查找顺序是“当前作用域 → 外层 → … → 全局”。找不到就会报错。

预期：var 版本通常输出 3 个相同数字；let 版本输出 0、1、2。
