# 前端自学 Day1-Day30 日计划表

> 节奏建议：工作日 1.5-2h，周末 3-4h。  
> 每天固定结构：30 分钟学习 + 60 分钟实战 + 20 分钟复盘。

## Day1-Day10（HTML/CSS/响应式/DOM基础）

| Day | 学习重点 | 实战任务 | 验收标准 |
|---|---|---|---|
| Day1 | HTML 语义化、页面结构 | 个人主页基础结构（header/main/section/footer） | 页面结构清晰，只有一个 `h1` |
| Day2 | 盒模型、间距管理、`border-box` | 卡片化页面、表单排版 | 无 `<br>` 调间距，布局不溢出 |
| Day3 | Flex 布局 | 导航 + 卡片流 + footer 响应式 | 能解释主轴/交叉轴，布局可换行 |
| Day4 | Position 定位 | sticky 导航 + absolute 角标 + fixed 回顶 | 能说清四种定位的适用场景 |
| Day5 | Responsive 响应式 | 375/768/1024 三档布局切换 | 无横向滚动，手机可读可点 |
| Day6 | DOM 选择与事件 | 点击按钮切换主题、更新文案 | 能使用 `querySelector` + `addEventListener` |
| Day7 | 列表渲染基础 | JS 动态添加/删除卡片 | 可新增/删除，DOM 更新正确 |
| Day8 | 表单与基础校验 | 输入校验 + 错误提示 | 空值/格式错误能拦截并提示 |
| Day9 | 本地存储 `localStorage` | 保存待办列表，刷新不丢 | 刷新后数据仍存在 |
| Day10 | 小复盘日 | 做一个“可交互 Todo”整合版 | 具备新增/删除/筛选/持久化 |

## Day11-Day20（工程化 + Vue3 入门）

| Day | 学习重点 | 实战任务 | 验收标准 |
|---|---|---|---|
| Day11 | JavaScript 数组与对象进阶 | 用真实数据结构重构 Todo（含筛选/统计） | 能独立使用 `map/filter/reduce` 完成数据处理 |
| Day12 | npm 与 Vite | 初始化 Vite 项目并跑通 | 能独立 `npm install` + `npm run dev` |
| Day13 | ES Module | 拆分 JS 文件并模块化导入 | 代码按功能拆分，主文件更清晰 |
| Day14 | TypeScript 入门 | 给简单数据结构加类型 | 能写 interface/type，消除明显 any |
| Day15 | Vue3 组件基础 | 创建组件并拆分页面 | 至少拆出 3 个可复用组件 |
| Day16 | 响应式状态 | `ref/reactive/computed` 练习 | 能解释 ref vs reactive |
| Day17 | 组件通信 | props + emit | 子组件可触发父组件更新 |
| Day18 | 组件库入门（Element Plus） | 安装并接入 Element Plus，替换按钮/输入框/卡片 | 能独立引入并渲染至少 3 个组件 |
| Day19 | 组件库表单与表格 | 用 `el-form` + `el-table` 重做一个列表页 | 表单校验和表格展示可用 |
| Day20 | Vue + 组件库小练习整合 | 完成一个“任务管理”Vue版（组件库UI） | 增删改查流程可用，页面风格统一 |

## Day21-Day30（Vue进阶 + API + 上线）

| Day | 学习重点 | 实战任务 | 验收标准 |
|---|---|---|---|
| Day21 | Vue Router | 首页/列表/详情三页路由 | 路由切换正常，有 404 页面 |
| Day22 | Pinia 状态管理 | 把共享状态迁到 store | 跨页面状态一致 |
| Day23 | HTTP 请求（Axios） | 封装请求模块 | 有基础错误处理 |
| Day24 | Mock 数据联调 | 列表查询 + 新增提交 | 能完成一次完整请求闭环 |
| Day25 | CRUD 实战 | 新增、编辑、删除、分页 | 交互完整且状态更新正确 |
| Day26 | 登录态与权限基础 | token 存储 + 路由守卫 | 未登录不能访问业务页 |
| Day27 | 组件库进阶与主题定制 | 配置主题色、统一按钮/表单风格 | 全局样式一致，主题可配置 |
| Day28 | 性能与代码整理 | 懒加载、拆组件、清理重复代码 | 首屏更快，结构更清晰 |
| Day29 | 部署上线 | 部署到 Vercel/Netlify | 获得可访问线上链接 |
| Day30 | 总结与展示 | 完成 README + 演示说明 | 可独立讲清项目方案与难点 |
